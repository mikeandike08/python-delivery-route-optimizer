Wondering how to use this program??


Pretty simple!


  1. First, make sure to have to have the necessary requirements, check the github for info
  2. If you want a quick and easy experience, import a csv of your addresses and you can get going!
  3. Don't worry! If not, you can simply type your addresses in the box, only use commas to seperate addresses
  4. VERY IMPORTANT make sure to include the city name in the address ex: 123 john doe city, NO commas
  5. Once you submit and everything is okay you can calculate the route, either by default(will not loop back) or use the checkbox for looping(utilizes TSP)
